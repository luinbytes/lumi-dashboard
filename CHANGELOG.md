# Changelog

All notable changes to Lumi Webpanel will be documented in this file.

## [Unreleased]

## [2.1.0] - 2026-01-31

### Added
- Improved error handling with try-catch blocks
- Better keyboard navigation for all interactive elements
- Enhanced accessibility with ARIA labels
- Clear visual feedback for all actions
- Toast notification system for user feedback
- Better mobile responsiveness
- Performance optimizations for animations

### Changed
- Refactored event listeners for better memory management
- Improved code organization with logical sections
- Better variable naming throughout
- Enhanced CSS for better contrast

### Fixed
- Memory leak in event listeners
- Focus management issues
- Expression cycling on mobile devices
- Theme toggle persistence on some browsers

### Security
- Sanitized all user inputs before processing
- Added CSP meta tag for security

## [2.0.0] - 2026-01-31

### Added
- Command input with history navigation
- 3 new expressions (excited, confused, sleeping)
- Sound effects toggle
- Animation speed controls (25% - 300%)
- 2 new stat cards (errors, responses)
- PNG screenshot export
- Activity log types with color coding
- Clear log button
- Footer links

### Improved
- Better keyboard navigation
- Enhanced accessibility
- Improved command handling
- More responsive design
- Better localStorage persistence

### Fixed
- Expression cycling issues
- Theme toggle persistence
- Activity log formatting
